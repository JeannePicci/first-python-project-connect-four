# Connect Four (First Python Project)

This repository contains one of my very first Python projects, originally created in 2023 while I was learning programming.

The goal of this project was to learn:
- Python fundamentals
- 2D lists
- Functions
- Event-driven programming
- Building a graphical interface with Tkinter

Although the code does not reflect my current programming practices, I chose to keep it as it was to illustrate my learning journey.
